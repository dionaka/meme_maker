# Meme Maker 插件 🎨  

---

## 📘 简介 | Introduction

**Meme Maker** 是一个基于 [AstrBot](https://docs.astrbot.app) 的图片合成插件。  
用户只需输入 `/add  /add1 /add2` 并发送一张图片，插件就会自动将该图片放置到模板的指定区域上，生成个性化梗图。


---

## ✨ 功能特性 | Features

- 🚀 支持 `/add  /add1 /add2` 指令启动图片制作流程  
  
- 🖼️ 自动检测用户发送的图片  

- 📏 智能缩放 + 居中裁剪，完美适配模板区域  

- 💾 模板文件本地加载，运行稳定可靠  

- 🧩 一键打包，支持 WebUI 上传安装
